Herramientas de IA

Existe una alta gama de diversidad de aplicaciones empleadas con IA. Las cuales pueden ser clasificadas en un top 3 de acuerdo a la necesidad:

  • Video

    • Para una creación y edición de vídeo mucho más rapida existe una variedad aplicaciones, entre las que destacan

    1. Supercreator.ai: http://supercreator.ai/
    2. Tavus: https://www.tavus.io/
    3. Windsor.io: https://www.windsor.io/
       <

  • Imagenes

    • Para una creación de arte e imagenes a partir de texto se tienen herramientas como:

    1. Stokimg.ai: https://stockimg.ai/
    2. Midjourney: https://www.midjourney.com//
    3. Dreamer: https://deepdreamgenerator.com/
      

  • Texto

    • Hoy existen aplicaciones de IA para generar texto y traducciones, donde destacan

    1. Notion: https://www.notion.so/es-es
    2. Jasper: https://www.jasper.ai/
    3. DeepL: https://www.deepl.com/es/translator
      

  • Audio

    • Para generación asi como edición de audio, musica y sonidos en minutos se tienen:

    1. WhisperMemos: https://whispermemos.com
    2. Soundful: https://soundful.com/
    3. Steno: https://steno.ai/
      

  • Investigación

    • Para los proyectos de Investigación se puede hacer uso de:

    1. Bearly: https://bearly.ai/
    2. Scholarcy: https://www.scholarcy.com/
    3. Frase.io: https://www.frase.io/
      

  • Presentaciones

    • Tener tu presentación escolar o laboral puede ser mas sencillo con

    1. Slidesai: https://www.slidesai.io/es
    2. Otterai: https://otter.ai/
    3. Murfai: https://murf.ai/
      

  • Diseño

    • Para diseñar logotipos e interfaces de usuario existe:

    1. Looka: https://looka.com/
    2. Gaileo: https://www.usegalileo.ai/
    3. Uizard: https://uizard.io/
      

  • Conversacion

    • Las IA's mas famosas que ademas de servir para conversar, pueden hacer muchas de las necesidades anteriormente comentadas:

    1. ChatGPT: https://chatgpt.com/
    2. Gemini: https://gemini.google.com/app?hl=es
    3. Copilot: https://copilot.microsoft.com/
